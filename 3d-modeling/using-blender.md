# Using [Blender](https://www.blender.org/)

I generally use Blender for mocks and basic 3d printing, so this guide serves as notes for my work.

**Basic navigation and shortcuts:**

- x -- Delete selected object
- t -- Show/Hide tool-shel

**Adding objects to the scene**

New objects are added at the point of the cursor. Immediately after the oject is created, you can edit the properties of this primitive.
